# Structuro Obsidian vault

Open deze map als vault in Obsidian. Staat in git naast de app-repo.

## Setup (10 min)

1. Obsidian → Open folder as vault → selecteer deze map
2. Geen plugins eerste 2 weken
3. Open `Structuro Home.md`

## Sync opties

- **Git (aanbevolen):** vault zit in repo, commit notes met code
- **iCloud:** symlink of kopie buiten repo (niet aanbevolen voor doctrine-sync met Cursor)

## Flow

- Ruwe gedachten → note hier
- NotebookLM batch-research → export als note
- Gemini → ClickUp taken uit notes
- Cursor leest `../STRUCTURO_CONTEXT.md` en linked notes

Zie `../structuro-actieplan-2026-06.md`.
